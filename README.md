# تطبيق أذكار الصباح والمساء — نسخة أندرويد

هذا المجلد جاهز لتحويل تطبيق الويب (`www/index.html`) إلى تطبيق أندرويد حقيقي (APK)
عبر GitHub Actions، دون الحاجة لتثبيت Android Studio على جهازك.

## الخطوات

### 1) إنشاء مستودع على GitHub
1. اذهب إلى github.com وأنشئ حسابًا إن لم يكن لديك.
2. اضغط **New repository**، وسمِّه مثلاً `adhkar-app`، واجعله **Public** أو **Private**.
3. لا تُضِف README أو .gitignore عند الإنشاء (موجودان هنا مسبقًا).

### 2) رفع هذا المجلد إلى المستودع
من جهازك، افتح الطرفية داخل هذا المجلد ونفّذ:

```bash
git init
git add .
git commit -m "أول نسخة من تطبيق أذكار الصباح والمساء"
git branch -M main
git remote add origin https://github.com/USERNAME/adhkar-app.git
git push -u origin main
```

استبدل `USERNAME` باسم حسابك على GitHub.

> بديل بلا سطر أوامر: اسحب كل الملفات والمجلدات إلى صفحة المستودع في متصفحك
> عبر زر **Add file → Upload files**.

### 3) البناء التلقائي
بمجرد الرفع، سيعمل GitHub Actions تلقائيًا (الملف موجود في
`.github/workflows/build-apk.yml`) وسيقوم بـ:
- تثبيت الاعتماديات
- مزامنة ملفات الويب مع مشروع أندرويد
- بناء ملف APK (نسخة Debug)

لمتابعة التقدّم: افتح تبويب **Actions** في صفحة المستودع، ستجد عملية باسم
"Build Android APK" قيد التشغيل (تستغرق عادة 3-5 دقائق).

### 4) تنزيل ملف APK
بعد اكتمال العملية بنجاح (علامة ✔ خضراء):
1. افتح العملية المكتملة من تبويب **Actions**.
2. انزل إلى قسم **Artifacts** أسفل الصفحة.
3. نزّل الملف المضغوط باسم `adhkar-app-debug-apk` — يحتوي على `app-debug.apk`.
4. انقل الملف إلى هاتف أندرويد وثبّته (قد يطلب منك تفعيل خيار
   "تثبيت من مصادر غير معروفة" في الإعدادات، لأن التطبيق ليس من متجر Google Play).

### تحديث محتوى التطبيق لاحقًا
عدّل الملف `www/index.html` (نفس ملف الويب)، ثم:

```bash
git add .
git commit -m "تحديث المحتوى"
git push
```

سيُعاد بناء APK تلقائيًا بالنسخة الجديدة.

## ملاحظات
- هذه نسخة **Debug** مناسبة للتجربة والاستخدام الشخصي والمشاركة المباشرة (Sideload).
  لنشر التطبيق على متجر Google Play لاحقًا يلزم توقيعه بمفتاح إصدار (Release
  keystore) وهذه خطوة إضافية عند الحاجة إليها.
- اسم التطبيق والمعرّف الحالي: **أذكار الصباح والمساء** / `com.nesaj.adhkar` —
  يمكن تغييرهما من `capacitor.config.ts` ثم إعادة تشغيل `npx cap sync android`.
- أيقونة التطبيق حاليًا هي الأيقونة الافتراضية لـ Capacitor. لتخصيصها، ضع صورة
  مربعة (1024×1024 تقريبًا) وحوّلها بأداة مثل
  https://icon.kitchen ثم استبدل الملفات داخل
  `android/app/src/main/res/mipmap-*/`.
